PHP-reference scanner

Click "Start!" to beginn scanning the PHP-reference-manual. It expects the manual in the folder "phpman/" and in the form "Many HTML files" (see Documentation downloads).
As soon as PHPCheck has scanned the manual, it knows all (okay, nearly all ;)) builtin functions and classes including its return- and parameter-types. Therefore it can also help you when you're using builtin-stuff. Additionally it collects since when a function exists or since when it does no longer exist. With this information it can tell you the minimal and maximal PHP-version that is required for your application.
